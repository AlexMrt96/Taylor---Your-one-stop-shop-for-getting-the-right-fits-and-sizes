# Taylor---Your-one-stop-shop-for-getting-the-right-fits-and-sizes
Taylor — Size Comparator is a browser-based tool that helps you compare clothing sizes across different brands and retailers side by side.
Core idea: Sizing is inconsistent across brands — a Medium at Zara fits differently than a Medium at Uniqlo. Taylor solves this by letting you place multiple items into comparison slots and visualising their actual measurements as scaled silhouettes, so you can see the differences at a glance.
How it works:
The interface is split into two areas. On the left, a sidebar lists a catalogue of size cards organised by clothing category — Tops, Bottoms, Shoes, Jackets, and Dresses. Each card shows the brand, item name, size label, and key measurements. You simply drag a card from the sidebar into one of up to four comparison slots on the right.
Once dropped into a slot, each item renders as an SVG silhouette — a shirt shape, trouser shape, shoe, jacket, or dress — proportionally scaled to its real dimensions. This makes it immediately obvious when one brand's "Large" is significantly wider or longer than another's.
Key features:

Multi-slot comparison — compare up to 4 items simultaneously
Visual silhouettes — garment shapes drawn from actual measurements (chest, waist, hip, inseam, sleeve length, etc.)
Search & filter — find items by name or brand, filter by retailer or category
Favourites — star frequently used sizes for quick access in a dedicated tab
Custom sizes — add your own items through a modal form, entering measurements manually
Export — save a PNG snapshot of the current comparison view

Who it's for:
Taylor is useful for online shoppers tired of guessing between sizes, stylists managing multiple clients, or anyone building a wardrobe across several brands who wants a clear, visual reference rather than a wall of numbers in a size chart.
The entire app runs in a single HTML file with no backend, no login, and no dependencies — just open it in a browser and start comparing.
